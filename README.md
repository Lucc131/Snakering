# Snakering

Jogo da cobrinha classico em **HTML, CSS e JavaScript**, com tema de cobranca: a cobrinha cresce ao coletar **dinheiro** em vez de frutas.

## Acesso rapido

- Repositorio: https://github.com/Lucc131/Snakering
- Jogo (GitHub Pages): https://lucc131.github.io/Snakering/

## Tecnologias

- HTML5 Canvas
- CSS3
- JavaScript Vanilla
- Tailwind CSS (CDN)
- GSAP (animacoes)

## Estrutura do projeto

```text
Snaker files/
├── index.html
├── html/
│   └── index.html
├── css/
│   └── style.css
└── js/
    └── game.js
```

## Como jogar

- `Setas` ou `WASD` para mover
- `Espaco` para reiniciar apos perder

## Rodar localmente

1. Abra o arquivo `index.html` da raiz do repositorio no navegador.
2. Ele redireciona automaticamente para o jogo.
